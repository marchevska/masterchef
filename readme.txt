Структура каталогов

dist - дистрибутив
dist/current - текущая версия дистрибутива (ничего лишнего)
dist/release - запакованные релизные версии, разбивка на демо/полную и по языкам (англ/рус)
dist/current/abc english - файлы, относящиеся к анлийской локализации
dist/current/abc russian - файлы, относящиеся к русской локализации
docs - документы и спецификации
graphics - исходники графики - в корне лежат файлы, относящиеся к английской версии
graphics/abc russian - исхдники графики для русской версии
references - образцы картинок для продуктов, рецептов и интерьеров
py - исходники на питоне           


Для переключения между языками запустить скрипт dist/currrent/abc english.bat или dist/currrent/abc russian.bat
